Projekty

Všetky moje terajšie práce a projekty z prvého semestru vysokej školy nájdete práve tu.

1. Nastavení prostředí, základy C

"Vytvořte program, který bude "rozměňovat" peníze. Na začátku programu (ve funkci main) si vytvořte celočíselnou proměnnou, do které uložte hodnotu (číslo), které bude mít váš program za úkol rozměnit na jednotlivé bankovky. Hodnotu do proměnné uložte "natvrdo", nemusíte ji načítat ze vstupu/terminálu/klávesnice za běhu programu."

C GCC
Kód

2. Cykly a podmínky

"Vaším úkolem bude vytvořit program, který bude umět vypisovat různé geometrické tvary na výstup. Při spuštění program načte ze standardního vstupu (z terminálu) tři čísla (obrazec, A a B). V závislosti na hodnotě čísla obrazec poté program vykreslí daný geometrický tvar, jehož tvar bude ovlivněn hodnotami parametrů A a B."

C GCC
Kód

3. Funkce

"Cílem cvičení je vytvořit program, ve kterém si procvičíte využití funkcí (v pozdějších úlohách budete funkce používat už automaticky v podstatě neustále). Snažte se tedy svůj kód rozumně rozdělit do funkcí, abyste kód, který se dá znovu využít, neměli v programu neustále zkopírovaný. Identifikujte často se opakující kusy kódu (vykreslení čáry, vykreslení čtverce atd.) a udělejte z nich funkce."

C GCC
Kód

4. Ukazatele

"Vytvořte program, který bude počítat histogram čísel. Histogram bude mít 9 "košů" a bude zaznamenávat četnost jednotlivých hodnot, které budou v rozsahu zadaném na vstupu programu."

C GCC
Kód

5. Dynamická alokace paměti

"Vytvořte program, který vytvoří dvourozměrné plátno/mřížku/obrázek, na kterém se budou pohybovat a kreslit želvičky. Uživatel aplikace bude příkazy ovládat pohyb a kreslení."

C GCC
Kód

6. Práce s textem

"Vytvořte program, který bude provádět manipulaci s textem. Program načte sadu řádků se slovy, každé slovo ze vstupu "znormalizuje", vypíše znormalizované slova, a také statistiku."

C GCC
Kód

7. Struktury

"Vytvořte program, který načte sadu záznamů o hodnotách akcií. Informace a statistiky o akciích poté program vypíše na standardní výstup ve formě HTML stránky."

C GCC
Kód

8. Textová práce se soubory

"Vytvořte program který obdrží cestu ke vstupnímu souboru a hledaný text pomocí parametrů příkazové řádky. Poté projde soubor řádek po řádku, a vypíše všechny řádky s hledaným textem."

C GCC
Kód
